
# Temperature Converter in Java

A simple console-based Java application that converts temperatures between Celsius, Fahrenheit, and Kelvin. I built this project to practice basic Java concepts, object-oriented programming (OOP), and user input handling.

## What it does

This application allows users to easily convert temperatures between Celsius, Fahrenheit, and Kelvin using clean logic and standard scientific formulas. It includes built-in exception handling, which means the program will not crash even if a user types text or invalid characters instead of numbers. Instead, it gently asks for the correct input, ensuring a smooth command-line experience with accurate decimal results.

## Formulas Used

* **Celsius to Fahrenheit:** (Celsius * 9/5) + 32
* **Celsius to Kelvin:** Celsius + 273.15
* **Fahrenheit to Celsius:** (Fahrenheit - 32) * 5/9

## Tech Used

* **Language:** Java
* **Concepts:** Classes & Objects, Scanner class for input, Exception Handling, Loops.
